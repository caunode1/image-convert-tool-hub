# 이미지 변환 툴 허브 브리프 v01

## 프로젝트명
이미지 변환 툴 허브

## 목표
- 일반 사용자가 검색해서 바로 쓰는 실용형 이미지 변환/압축 사이트 구축
- 애드센스 승인 가능한 정보형 툴 사이트 구조 확보
- 빠르게 MVP를 공개하고, 이후 SEO/콘텐츠/툴 페이지를 확장

## 핵심 사용자
- 이미지 파일 형식을 바꾸려는 일반 사용자
- WEBP/JPG/PNG 변환이 필요한 사용자
- 이미지 용량 줄이기/크기 조절이 필요한 사용자
- 설치 없이 웹에서 빠르게 해결하려는 사용자

## MVP 범위
### 툴
- WEBP → JPG
- WEBP → PNG
- JPG → PNG
- PNG → JPG
- JPG → WEBP
- PNG → WEBP
- 이미지 압축
- 이미지 크기 조절

### 정보 문서
- WEBP/JPG/PNG 차이 설명
- 이미지 변환 시 화질이 달라지는 이유
- 파일을 서버에 저장하지 않는 처리 원칙
- 이미지 용량 줄이기 가이드
- 모바일/카톡/아이폰 저장 이슈 가이드

## 사이트 방향
- 포맷 이름만 바꾸는 장난감이 아니라, 실제 브라우저 내 포맷 변환 툴로 설계
- 가능하면 클라이언트 사이드 처리 중심으로 가서 개인정보/파일 보관 리스크 최소화
- 툴 + 설명 문서 + FAQ + 정책 + 문의 페이지를 함께 둔 정보형 서비스 구조

## 수익화 방향
- 초기 목표는 애드센스 승인
- 승인 전까지는 얇은 랜딩보다 가이드/정책/문의/방법론 구조를 우선 강화
- 승인 후 광고 배치는 사용성 저해 없는 범위에서 진행

## 빠른 실행 우선순위
1. 프로젝트 구조/브랜드/정보 구조 확정
2. 이미지 변환 기본 툴 2~3개 MVP 구현
3. 정책/문의/가이드 문서 구축
4. SEO/정적 페이지 구조 강화
5. 애드센스 심사 준비
