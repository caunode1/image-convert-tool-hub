# Image Convert Tool Hub — AdSense Application Checklist

작성일: 2026-03-28
대상 사이트: Image Convert Tool Hub

## 전제
AdSense 승인은 구글 심사 결과라 **무조건 승인 보장**은 불가능하다.
이 체크리스트는 승인 확률을 높이기 위한 실무용 점검표다.

## 현재 강점
- [x] 도구 목적이 분명함
- [x] 실제 기능이 동작함
- [x] 가이드 콘텐츠 다수 확보
- [x] 소개 / 방법론 / 개인정보처리방침 / 이용안내 / FAQ / 문의 페이지 존재
- [x] 문의 이메일 노출됨
- [x] robots.txt / sitemap.xml 존재
- [x] 모바일 기본 가독성 확보
- [x] 얇은 광고 랜딩보다 유틸리티 + 설명 사이트 구조로 보강 완료

## 아직 사람이 직접 확인해야 하는 것
- [ ] 최종 커스텀 도메인 연결
- [ ] Search Console 등록 및 sitemap 제출
- [ ] 운영 이메일 실제 수신 테스트
- [ ] 신청 직전 broken link 최종 점검
- [ ] 필요시 실기기에서 모바일 체감 점검

## 콘텐츠 품질 체크
- [x] 원본성 있는 설명 콘텐츠가 있는가
- [x] 도구 사용법과 제한 사항을 숨기지 않는가
- [x] 단순 문장 반복/자동 생성 느낌이 과하지 않은가
- [x] 한두 페이지에만 의존하지 않고 여러 가이드가 있는가
- [x] 사용자에게 실제 도움 되는 문서가 있는가

## 사이트 구조 체크
- [x] 홈에서 사이트 목적이 바로 보이는가
- [x] 도구 페이지 접근이 쉬운가
- [x] 가이드 허브가 있는가
- [x] 정책/문의 페이지 접근이 쉬운가
- [x] footer에 기본 정보가 있는가

## 정책/신뢰 체크
- [x] 개인정보처리방침 존재
- [x] 이용안내 존재
- [x] 문의 페이지 존재
- [x] FAQ 존재
- [x] 처리 방식/제한 사항을 문서로 안내함
- [x] 업로드 파일 보관/처리 원칙을 설명함

## 기술 체크
- [x] canonical 메타 동작
- [x] robots.txt 존재
- [x] sitemap.xml 존재
- [x] 주요 페이지 라우팅 정상
- [x] Pages 배포 정상
- [ ] 커스텀 도메인 canonical 반영
- [ ] Search Console 인덱싱 확인

## 애드센스 신청 직전 최종 체크
### 필수
- [ ] 최종 운영 도메인 확정

- [ ] 핵심 페이지 인덱싱 경로 확보
- [ ] 홈 / 가이드 / 정책 / 문의 페이지 다시 읽어도 얇지 않음
- [ ] AdSense 코드 삽입 가능한 HTML 접근권 확보

### 권장

## 지금 기준 판단
### 현재 상태
**신청 준비도: 꽤 높아진 상태, 하지만 도메인/인덱싱 단계가 남아 있음**

### 가장 중요한 남은 3개
1. **커스텀 도메인 연결**
2. **Search Console 등록 + sitemap 제출**
3. **신청 직전 최종 점검**

## 신청 순서 추천
1. 커스텀 도메인 연결
2. canonical / sitemap / robots 새 도메인 기준 확정
3. Search Console 등록
4. sitemap 제출
5. 핵심 URL 수동 검사
6. 2~7일 정도 상태 관찰
7. AdSense 신청

## 지금 단계 한 줄 정리
이 사이트는 이제 ‘기능만 있는 얇은 도구 사이트’보다 **설명·정책·신뢰 요소가 붙은 실사용 유틸리티 사이트**에 더 가깝다. 다음 승부처는 코드보다 **도메인, 인덱싱, 운영 흔적**이다.
