# 이미지 변환 툴 허브

## 한 줄 정의
일반 사용자가 검색해서 바로 쓰는 이미지 포맷 변환·압축·리사이즈 툴 허브

## 목표
- 애드센스 승인 가능한 실용형 툴 사이트 구축
- 일반 사용자 대상 검색형 트래픽 확보
- 빠른 MVP 공개 후 툴/가이드/정책 페이지 확장

## 핵심 사용자
- WEBP/JPG/PNG 변환이 필요한 일반 사용자
- 이미지 용량 줄이기/크기 조절이 필요한 사용자
- 설치 없이 웹에서 바로 해결하고 싶은 사용자

## MVP 기능
- WEBP → JPG
- WEBP → PNG
- JPG → PNG
- PNG → JPG
- JPG → WEBP
- PNG → WEBP
- 이미지 압축
- 이미지 리사이즈

## 콘텐츠/문서
- WEBP/JPG/PNG 차이 설명
- 화질 손실이 생기는 이유
- 브라우저 내 처리 / 서버 비보관 원칙
- 이미지 용량 줄이기 가이드
- 모바일/카톡/아이폰 파일 이슈 가이드

## 운영 원칙
- 가능하면 브라우저 내 처리 우선
- 업로드 파일 서버 저장 최소화 또는 비보관 원칙 명확화
- 툴 + 설명 문서 + 정책 + 문의 채널 동시 운영
- 얇은 랜딩보다 실제 사용 가치가 있는 페이지 우선
